# Vibe Studio — MVP

바이브 코더용 개발 스튜디오 UI/UX 프로토타입. 와이어프레임 기반 3열 레이아웃.

## 레이아웃
- **좌측**: 파일시스템(접이식) + 유틸리티(로컬 검수·스캔·MCP·토큰)
- **중앙**: 상단 탭(에이전트 CoT / 라이브 웹 / 대시보드 / PDF / CSV) → 메인 뷰포트 → 툴바(질문모드 + 세션 1·2·3·+) → 채팅 입력
- **우측**: 질문/답변 대화 트랜스크립트

## 핵심 기능 (구현됨)
- **질문 모드(✋)**: 툴바의 손 아이콘을 켜고 라이브 웹 프리뷰의 요소를 클릭하면,
  그 부분의 이름("📍 URL 입력 필드")이 칩으로 프롬프트에 자동 삽입된다.
  사용자에겐 자연어 라벨만 보이고, 내부적으로는 요소의 selector(위치)까지 함께 전달된다.
- **세션 전환/추가**: 하단 1·2·3 세션 탭, `+` 로 추가.
- **탭 전환 뷰포트**: 라이브 웹(iframe), 에이전트 CoT, 대시보드, CSV, PDF.
- **접이식 좌/우 패널**.

## 실행
```bash
pnpm install   # 또는 npm install
pnpm dev       # http://localhost:5180
```

## 데스크톱 앱 포팅
이 웹 프론트엔드를 그대로 Tauri 또는 Electron 쉘에 넣으면 데스크톱 앱이 된다.
질문 모드의 요소-피킹은 실제 dev 서버 프리뷰에 스크립트를 주입하는 방식으로 확장한다.

## 기술
React 18 + Vite. 상태는 로컬 컴포넌트 상태(목업 데이터). 백엔드/에이전트 연동은 후속.
